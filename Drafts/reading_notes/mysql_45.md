 # Advice

要多写SQL，培养自己对SQL语句执行效率的感觉。以后再写或者建 索引的时候，知道这个语句执行下去大概的时间复杂度，是全表扫描还是索引扫描、是不是需要 回表，在心里都有一个大概的概念。
